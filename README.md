# Campus Student Support & Appointment Booking System

This project is a full-stack web application developed as part of the Master of Computer Applications (MCA) curriculum. The system is designed to help students book campus-related support services and manage appointments efficiently.

The application demonstrates complete full-stack development using open-source technologies, including frontend design, backend logic, database integration, and user authentication.

---

## Project Objective

The main objective of this project is to design and develop a web-based system that allows students to:
- Register and log in securely
- View available campus support services
- Book appointments for selected services
- Track booking status

The system also provides an admin interface to manage services and student bookings.

---

## Features

### Student Module
- Student registration and login
- View available campus services
- Book service appointments
- View booking history and status

### Admin Module
- Admin login
- Manage services
- View student bookings
- Approve or reject appointments

---

## Technology Stack (Open Source)

- **Frontend:** HTML5, CSS3, Bootstrap, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  
- **Server:** Apache (XAMPP)

All technologies used in this project are open-source and comply with MCA project guidelines.

---

## Database Features

- Properly normalized relational database
- Multiple tables with primary and foreign keys
- CRUD operations (INSERT, SELECT, UPDATE, DELETE)
- JOIN queries for data retrieval
- Session-based authentication
